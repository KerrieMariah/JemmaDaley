# Jemma Daley – Financial Advisor Website

A personal website for Jemma Daley, a financial advisor in Hong Kong specialising in wealth management for expats.

## Tech Stack

- **Next.js 15** with App Router
- **Tailwind CSS 4**
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Color Scheme

- Teal accent: `#3fdcc8`
- Navy: `#0a1428`
- White background theme

## Photos

Place photos in the `public/` directory. The site uses `jemma1.jpeg` through `jemma6.jpeg`.
